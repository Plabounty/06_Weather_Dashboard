# 06_Weather_Dashboard

Weather app that you type a city name in the search field and then shows you results for that city.

It shows you the Temperature, Humidity, Wind Speed, Clouds, and UV Index.

Saves search in local storage to access previous searches if need be.


Published web page: https://plabounty.github.io/06_Weather_Dashboard/
